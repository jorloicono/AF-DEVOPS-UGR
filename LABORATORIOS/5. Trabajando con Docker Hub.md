# Paso 1: Crear una cuenta en Docker Hub

1. Ve a [https://hub.docker.com](https://hub.docker.com)
2. Crea una cuenta o inicia sesión.
3. Desde el panel podrás administrar tus repositorios.

---

# Paso 2: Crear un repositorio (opcional)

1. En Docker Hub haz clic en **Create Repository**.
2. Nombre del repositorio: `tomcat-java-app`.
3. Visibilidad:

* 🔓 **Public**
* 🔒 **Private**

4. Haz clic en **Create**.

---

# Paso 3: Crear una aplicación Java para Tomcat

Aquí haremos una **aplicación web Java simple usando Servlets** que generará un `.war`.

---

## 1️⃣ Crear la estructura del proyecto

```bash
mkdir tomcat-java-app
cd tomcat-java-app
```

Estructura:

```
tomcat-java-app
│
├── src
│   └── main
│       ├── java
│       │   └── com/example/HelloServlet.java
│       │
│       └── webapp
│           └── WEB-INF
│               └── web.xml
│
└── pom.xml
```

---

# Paso 4: Crear el Servlet

Archivo:

```
src/main/java/com/example/HelloServlet.java
```

```java
package com.example;

import java.io.IOException;
import java.io.PrintWriter;

import jakarta.servlet.ServletException;
import jakarta.servlet.http.HttpServlet;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;

public class HelloServlet extends HttpServlet {

    @Override
    protected void doGet(HttpServletRequest request, HttpServletResponse response)
            throws ServletException, IOException {

        response.setContentType("text/html");

        PrintWriter out = response.getWriter();
        out.println("<h1>Hola desde Java + Tomcat + Docker</h1>");
    }
}
```

---

# Paso 5: Configurar `web.xml`

Archivo:

```
src/main/webapp/WEB-INF/web.xml
```

```xml
<web-app xmlns="http://xmlns.jcp.org/xml/ns/javaee"
         version="4.0">

    <servlet>
        <servlet-name>HelloServlet</servlet-name>
        <servlet-class>com.example.HelloServlet</servlet-class>
    </servlet>

    <servlet-mapping>
        <servlet-name>HelloServlet</servlet-name>
        <url-pattern>/hello</url-pattern>
    </servlet-mapping>

</web-app>
```

---

# Paso 6: Crear `pom.xml`

Esto permite generar el **archivo WAR**.

```xml
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0
         http://maven.apache.org/xsd/maven-4.0.0.xsd">

    <modelVersion>4.0.0</modelVersion>

    <groupId>com.example</groupId>
    <artifactId>tomcat-java-app</artifactId>
    <version>1.0</version>

    <packaging>war</packaging>

    <dependencies>
        <dependency>
            <groupId>jakarta.servlet</groupId>
            <artifactId>jakarta.servlet-api</artifactId>
            <version>5.0.0</version>
            <scope>provided</scope>
        </dependency>
    </dependencies>

</project>
```

---

# Paso 7: Compilar el proyecto

Ejecuta:

```bash
mvn clean package
```

Esto generará:

```
target/tomcat-java-app.war
```

---

# Paso 8: Crear el Dockerfile con Tomcat

Archivo **Dockerfile**

```Dockerfile
# Imagen oficial de Tomcat
FROM tomcat:10-jdk17

# Eliminar aplicaciones por defecto
RUN rm -rf /usr/local/tomcat/webapps/*

# Copiar el WAR a Tomcat
COPY target/tomcat-java-app.war /usr/local/tomcat/webapps/ROOT.war

# Exponer puerto
EXPOSE 8080
```

---

# Paso 9: Construir la imagen Docker

```bash
docker build -t tu_usuario_dockerhub/tomcat-java-app .
```

---

# Paso 10: Ejecutar el contenedor

```bash
docker run -p 8080:8080 tu_usuario_dockerhub/tomcat-java-app
```

Ahora puedes acceder a la aplicación en:

```
http://localhost:8080/hello
```

---
